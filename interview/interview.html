• What is React?

    React is a javascript library used to create single page application

  • What are the major features of React?

    1. Uses a Jsx syntax in order to allow user to write html in js
    2. It uses virtual dom instead of real dom in order to improve performance
    3. It allow users to create reusable components

  • How Virtual DOM works?

    When ever a underlying value is changed the entire UI component will be re rendered in virtual dom. Now th deference between the previous one and the new virtual dom is calculated,when the calculation is done then the real dom will be updated with only the changed values

  • What is the difference between Shadow DOM and Virtual DOM?

    The shadow dom browser concept primarily used for scoping variables and css in web and the virtual dom is a concept made up on js on top of browser apis
