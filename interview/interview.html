• What is React?

    React is a JavaScript library used to create single-page applications.

  • What are the major features of React?

    1. Uses a JSX syntax to allow writing HTML in JS
    2. Uses Virtual DOM instead of real DOM for better performance
    3. Allows creation of reusable components

  • How does Virtual DOM work?

    Whenever a value changes, the entire UI component is re-rendered in the Virtual DOM. Then the difference between the previous and the new Virtual DOM is calculated. After that, only the changed values are updated in the real DOM.

  • What is the difference between Shadow DOM and Virtual DOM?

    Shadow DOM is a browser feature used for scoping variables and CSS in web components, while Virtual DOM is a concept implemented in JavaScript libraries like React to optimize DOM updates.

  • What are controlled components?

    Controlled components are form inputs that are controlled by React. The state of the input is managed by the component's state, making it easy to access and manipulate input values.

  • What is HTML?

    HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines elements like headings, paragraphs, links, and more.

    which is elements that contains the meaning of the content and the structure of the html Document. These elements contains content that is related to their name or reflects their name
